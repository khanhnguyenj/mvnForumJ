1/ This package include Tomcat that has been bundled with MVN softwares
   (mvnForum/mvnAd/mvnCMS)
   
2/ For running, open command line console
   For Windows:
      cd bin
      startup.bat
   For Linux
      cd bin
      ./startup.sh
